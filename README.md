# MERN Task Manager

A full-stack Task Manager application built using the MERN stack.

## Features

- User Registration
- User Login with JWT Authentication
- Create Tasks
- View Tasks
- Mark Tasks as Completed
- Delete Tasks
- Protected Routes
- Responsive Dashboard UI

## Technologies Used

### Frontend
- React
- React Router DOM
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
